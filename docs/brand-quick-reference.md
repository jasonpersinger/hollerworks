# HOLLER.WORKS Quick Reference

## Brand Basics

- Display brand: `HOLLER.WORKS`
- Editorial/body use: `holler.works`
- Descriptor: `a moderated job board for tech and tech-adjacent work in Appalachia`

## Core Feel

- utilitarian but stylish
- grounded and internet-native
- old-web aware, not retro gimmicky
- regional, human, and credible

## Voice

- direct
- restrained
- useful
- human
- lightly rough-edged

Avoid:

- startup buzzwords
- empty optimism
- generic “future of work” language

## Casing Rules

- Use lowercase more freely in branding, social, launch, and editorial copy.
- Keep standard sentence case in UI, forms, system copy, and anything operationally sensitive.
- Avoid `Holler.Works` as the default style.

## Slash Device

Use `//` as a brand header or system-style opener.

Examples:

- `// appalachia`
- `// launching holler.works`
- `// new jobs this week`

Good social shape:

`// LAUNCHING HOLLER.WORKS`

`>> stay. build. connected.`

`appalachia’s dedicated tech job board is now live.`

## Supporting Language

- `// appalachia`
- `STAY.`
- `BUILD.`

Use sparingly and confidently.

## Palette

- Black: `#0E0E0E`
- Rust: `#F28927`
- Appalachia Green: `#39803E`
- Gray: `#AAAAAA`
- Light Gray: `#CCCCCC`

Rules:

- keep rust dominant over green
- keep black as the canvas
- do not casually introduce new accent colors

## Logo System

- pixel mascot is a real signature asset
- use the standardized wordmark-only asset when the full lockup is too tall or too busy
- keep it crisp and pixelated
- use on black or near-black backgrounds
- rust wordmark + green supporting line is the preferred treatment

Avoid:

- blurring or smoothing the mascot
- random recolors
- busy photo backgrounds

## UI Rules

Keep:

- rails
- borders
- monospace foundation
- visible structure
- obvious actions and states

Avoid:

- glossy SaaS polish
- oversized whitespace
- rounded, soft UI language
- generic illustration-heavy product pages

## Litmus Test

Ask:

1. Does this look like HOLLER.WORKS at a glance?
2. Does it keep the black/rust/green system?
3. Does it feel useful first and stylish second?
4. Does it sound human?
5. Does the old-web influence feel intentional?
