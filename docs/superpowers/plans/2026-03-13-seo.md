# SEO Implementation Plan

> **For agentic workers:** REQUIRED: Use superpowers:subagent-driven-development (if subagents available) or superpowers:executing-plans to implement this plan. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Switch from hash routing to History API routing and add meta tags + JSON-LD structured data so post URLs are real indexable paths and Google can show rich job listing cards.

**Architecture:** All changes in `index.html`. Tier 1 rewrites the router (getRoute, navigate, navigateBoard, event listener, back links). Tier 2 adds static meta tags to head, a resetMeta() helper called on every render, and dynamic title/OG/JSON-LD updates in renderPost().

**Tech Stack:** Vanilla JS History API (pushState/popstate), schema.org JobPosting JSON-LD, Open Graph meta tags. Netlify is already configured for SPA routing -- no config changes needed.

Find `getRoute()` (line 247):

```js

    function navigateBoard() {
      const params = new URLSearchParams();
      if (state.filter !== 'all') params.set('filter', state.filter);
      if (state.cat) params.set('cat', state.cat);
      if (state.remote) params.set('remote', '1');
```

- [ ] **Step 9: Add resetMeta() function**

Find the `render()` function opening (around line 320):


Insert `resetMeta()` definition before `render()`, and call it as first line inside `render()`:

```js
    function resetMeta() {
      document.title = 'holler.works // tech jobs & skills \u2014 appalachia';
      document.querySelector('meta[name="description"]').content =
        'holler.works -- tech jobs and skills board for Appalachia. No recruiters. Compensation required on every listing.';
      document.querySelector('meta[property="og:title"]').content =
        'holler.works // tech jobs & skills \u2014 appalachia';
      document.querySelector('meta[property="og:description"]').content =
        'holler.works -- tech jobs and skills board for Appalachia. No recruiters. Compensation required on every listing.';
      const ogUrl = document.querySelector('meta[property="og:url"]');
      if (ogUrl) ogUrl.remove();
      const jsonLd = document.getElementById('json-ld-jobposting');
      if (jsonLd) jsonLd.remove();
    }

    async function render() {
      resetMeta();
      const route = getRoute();
```

- [ ] **Step 10: Add dynamic meta + JSON-LD in renderPost()**

Find (line 798, inside renderPost, right after the post-not-found guard returns):

```js
        const p = snap.data();
        const date = p.createdAt?.toDate
```

Insert after `const p = snap.data();`:

```js
        // ── SEO: dynamic meta + JSON-LD ────────────────────────────────────
        document.title = p.title + ' -- holler.works';
        document.querySelector('meta[name="description"]').content =
          (p.description || '').slice(0, 160);
        document.querySelector('meta[property="og:title"]').content =
          p.title + ' -- holler.works';
        document.querySelector('meta[property="og:description"]').content =
          (p.description || '').slice(0, 160);
        const ogUrlMeta = document.createElement('meta');
        ogUrlMeta.setAttribute('property', 'og:url');
        ogUrlMeta.content = 'https://holler.works/post/' + id;
        document.head.appendChild(ogUrlMeta);

        if (p.type === 'need') {
          const locationAddr = {
            '@type': 'PostalAddress',
            addressLocality: p.locationCity || p.locationRegion || '',
            addressRegion:   p.locationState || '',
            addressCountry:  'US'
          };
          const schema = {
            '@context': 'https://schema.org',
            '@type': 'JobPosting',
            title: p.title,
            description: p.description +
              (p.compensation ? '\n\nCompensation: ' + p.compensation : ''),
            datePosted: p.createdAt?.toDate
              ? p.createdAt.toDate().toISOString()
              : new Date().toISOString(),
            hiringOrganization: {
              '@type': 'Organization',
              name: 'holler.works community',
              sameAs: 'https://holler.works'
            },
            jobLocation: { '@type': 'Place', address: locationAddr },
            ...(p.remoteFriendly ? {
              jobLocationType: 'TELECOMMUTE',
              applicantLocationRequirements: { '@type': 'Country', name: 'US' }
            } : {}),
            occupationalCategory: p.category || '',
            directApply: true,
            url: 'https://holler.works/post/' + id
          };
          const script = document.createElement('script');
          script.type = 'application/ld+json';
          script.id = 'json-ld-jobposting';
          script.textContent = JSON.stringify(schema);
          document.head.appendChild(script);
        }
        // ── end SEO ────────────────────────────────────────────────────────
```
