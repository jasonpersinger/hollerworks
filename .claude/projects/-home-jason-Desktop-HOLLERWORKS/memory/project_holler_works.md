Community tech job/skills board for Appalachia. Two-sided: "I need X" and "I can offer X."

**Why:** No one is serving the Appalachian tech/remote worker community with a focused job board. Holler is Appalachian slang for a hollow/valley — also means to call out.

**How to apply:** This is the primary active project. All work should follow the approved design spec at docs/superpowers/specs/2026-03-12-holler-works-design.md.
