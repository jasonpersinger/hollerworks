# HOLLER.WORKS Brand Assets

This folder is the canonical brand asset package for HOLLER.WORKS.

The goal is simple: when someone asks "which logo file should I use?", the answer should live here instead of in guesswork.

Rule of thumb:

- `master` = transparent source/master asset
- `primary` or `on-black` = default branded display export

## Logo Files

- `logo/hollerworks-lockup-primary.png`
  - Full primary lockup
  - Use when the brand needs to identify itself clearly on its own

- `logo/hollerworks-lockup-master.png`
  - Transparent master for the full lockup
  - Use for production/layout flexibility while preserving the official lockup

- `logo/hollerworks-wordmark-primary.png`
  - Wordmark-only logo
  - Use when the full lockup is too tall or busy but the brand still needs to read clearly

- `logo/hollerworks-wordmark-master.png`
  - Transparent master for the wordmark
  - Use when the brand name needs to sit cleanly inside a layout without a baked background

- `logo/hollerworks-mascot-master.png`
  - Canonical mascot mark
  - Transparent mascot master for avatars, favicons, UI accents, and mascot-only placements

- `logo/hollerworks-mascot-on-black.png`
  - Mascot display export on the approved black background
